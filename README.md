# Bank Cards Application

## Описание
Приложение для управления банковскими картами.  
Позволяет пользователям:
- Регистрироваться и авторизоваться
- Создавать и просматривать карты
- Переводить средства между картами
- Отображение номера карты через маску (Cipher)
- Безопасно работать с данными через Spring Security

---

## Технологический стек
- **Язык программирования:** Java 17+
- **Фреймворки:** Spring Boot, Spring MVC, Spring Security, Spring Data JPA
- **База данных:** PostgreSQL, H2 (для тестов)
- **Тестирование:** JUnit 5, Mockito

---

## Структура проекта
- **Controller** – обработка HTTP-запросов
- **Service** – бизнес-логика приложения
- **Repository** – работа с базой данных
- **Model/Entity** – сущности базы данных
- **DTO** – объекты для передачи данных между слоями
- **Security** – настройка аутентификации и авторизации
